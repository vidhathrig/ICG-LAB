This is the project for Interactive Computer Graphics Lab. 
This is a simple game made using 3js. The player controls the ball using the 4 arrow keys. The ball has to avoid the trees in its way and move forward. The score will be awarded for not colliding. Player has 3 lives for each game, i.e, he can collide with tree at most three times. After that it is game over.

Score and remaining life are displayed in the top left corner of the screen

Please run this project using mozilla firefox browser. In chrome game can be played but the ground texture will not be loaded into chrome.
